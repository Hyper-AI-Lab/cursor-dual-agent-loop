## Recommended follow-ups (manual / social)

These cannot be fully automated from the repo alone:

1. **Pin** this repository on the Hyper-AI-Lab org profile.
2. Set a **social preview** image in GitHub → Settings → Social preview (upload a PNG export of `docs/assets/banner.svg`).
3. Publish a short post: “How we spend Cursor Auto quota without babysitting every turn” linking here (X / LinkedIn / blog / HN “Show HN”).
4. Submit to curated lists when appropriate (awesome-cursor, awesome-ai-agents, etc.) — follow each list’s rules; don’t spam.
5. Cross-link from https://hyperailab.com/ and related product docs.
6. Cut annotated **Releases** (`v0.1.0`, …) — Release pages are indexed and shareable.
7. Enable **Discussions** for “Show your loop config” / Q&A (optional).
8. Keep the README’s first screen scannable; GitHub search weights README + description + topics heavily.
